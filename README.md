# EAIHAH_Network_for_Plant_Disease_Classification_and_RWLID_Dataset

This repository contains the implementation of EAIHAH-Net, a deep learning framework for plant disease classification in complex background environments, along with the RWLID dataset (Real-Field Wheat Leaf Image Dataset).
The RWLID dataset can be accessed using this Google Drive link (https://drive.google.com/drive/folders/1IzkOIFB_mE5HiQxZcGuUUyZmrlERSf-_?usp=drive_link).

Plant disease recognition in real agricultural scenes is challenging because of cluttered backgrounds, illumination variation, symptom similarity, and large intra-class and inter-class appearance changes. To address these issues, this work proposes EAIHAH-Net, which integrates frequency-aware analysis with hybrid feature learning to improve classification robustness under complex field conditions.

The EAIHAH-Net weights can be accessed using this Google Drive link (https://drive.google.com/drive/folders/1QMzrbYfFqvx7xRJGm3CjJUn_nWHjw0Fz?usp=drive_link).
