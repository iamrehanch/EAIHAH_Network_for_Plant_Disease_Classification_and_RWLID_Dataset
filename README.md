
This repository contains the implementation of EAIHAH-Net along with a new RWLID dataset.

Two datasets are publicly available:
Dataset1 (https://data.mendeley.com/datasets/ptz377bwb8/1).
Dataset2 (https://doi.org/10.7910/DVN/FJ1DM1).

Dataset3 (RWLID dataset) can be accessed using this Google Drive link (https://drive.google.com/drive/folders/1IzkOIFB_mE5HiQxZcGuUUyZmrlERSf-_?usp=drive_link).
The EAIHAH-Net weights can be accessed using this Google Drive link (https://drive.google.com/drive/folders/1QMzrbYfFqvx7xRJGm3CjJUn_nWHjw0Fz?usp=drive_link).
